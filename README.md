# Chrome Extension Currency Calculator

## Project Structure

* dist: Chrome Extension directory

## Setup

```
npm install
```

## Build

```
npm run build
```

## Build in watch mode

### terminal

```
npm run watch
```

## Load extension to chrome

Load `dist` directory


  https://chrome.google.com/webstore/detail/currency-calculator/pbhlpimiabffgiljfnopihjknkoemifb
